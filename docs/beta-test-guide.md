# Paddlio 4.0 Beta-Test Anleitung

## Ziel

Diese Beta soll zeigen, ob Paddlio im echten Vereinsalltag verstaendlich, stabil und nuetzlich ist. Bitte teste echte Ablaeufe und melde Fehler direkt über `Mehr > Feedback`.

## Tester

1. Einloggen.
2. Profil pruefen.
3. Training ansehen.
4. Anwesenheit setzen.
5. Aufgabe erledigen.
6. Nachricht testen.
7. Ergebnis anschauen.
8. Feedback senden.

## Trainer

1. Gruppe pruefen.
2. Training erstellen.
3. Aufgabe vergeben.
4. Anwesenheit pruefen.
5. Nachricht an Gruppe senden.
6. Ergebnis eintragen.
7. Feedback der Sportler ansehen.

## Admin

1. Nutzer pruefen.
2. Rollen pruefen.
3. Verein pruefen.
4. Beta-Check starten.
5. Feedback auswerten.
6. Beta-Tester-Status pflegen.

## Feedback schreiben

Bitte melden:

- Fehler oder Abstuerze
- unklare Texte
- abgeschnittene Buttons
- falsche Rollenrechte
- fehlende Daten
- langsame oder verwirrende Ablaeufe

Bitte immer die betroffene Seite und eine kurze Beschreibung angeben.

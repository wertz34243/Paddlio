# Desktop Layout

Desktop bleibt die vollständige Verwaltungsoberfläche.

## Vollständig auf Desktop

- Vereinsverwaltung
- Rollen und Rechte
- Import und Export
- Akademie-Redaktion
- große Analysen
- Wettkampfverwaltung
- Polar-Sync-Historie
- Admin- und Beta-Werkzeuge

Desktop nutzt Sidebar, Tabellen, Filter und dichte Bearbeitungsansichten.

# Phone Navigation

Die Smartphone-Version bleibt auf maximal fünf Hauptpunkte begrenzt:

1. Heute
2. Training
3. Analyse
4. Team
5. Mehr

## Mehr auf Phone

Die Mehr-Übersicht wird reduziert auf tägliche oder häufige Funktionen:

- Akademie
- Integrationen
- Profil
- Einstellungen
- Feedback
- Ziele
- Material
- Updates
- Coach, falls die Rolle berechtigt ist
- Beta, falls die Rolle berechtigt ist

Komplexe Verwaltungsfunktionen bleiben auf Tablet und Desktop besser erreichbar.

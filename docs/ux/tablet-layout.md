# Tablet Layout

Tablet ist die bevorzugte Arbeitsoberfläche für Trainer am Wasser.

## Zielbild

- Wochenplanung vollständig
- Kalender und Detailansicht nebeneinander
- Sportlerliste und Feedback nebeneinander
- Anwesenheit schnell erfassen
- Akademie-Inhalte zuweisen
- kleinere Importvorschauen prüfen

Portrait darf kompakter werden. Landscape soll Split-Layouts bevorzugen.
